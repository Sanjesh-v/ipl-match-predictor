# IPL Match Predictor

## Overview

Machine learning web application that predicts the winner of an IPL match based on:

- Team 1
- Team 2
- Venue
- Toss Winner
- Toss Decision

## Tech Stack

Frontend:
- React
- Axios
- Vite

Backend:
- FastAPI
- Python

Machine Learning:
- Scikit-Learn
- Random Forest Classifier
- Pandas

## Features

- Real-time predictions
- Interactive UI
- Confidence score visualization
- Team validation
- REST API architecture
